常见问题

它能脱离电脑单独聊天吗+

不能。它需要连接电脑端或者云服务器 Hermes Studio。ESP32 负责硬件交互，AI 能力由 Hermes Studio 和你配置的服务提供。

买来后需要配置什么+

需要给设备配 Wi-Fi，在 Hermes Studio 中完成配对，并配置语音识别、语音合成和模型服务。

声音适合什么场景+

适合桌面近距离听，定位是语音提示和近场对话反馈，不是大功率音箱或远场智能音箱。

可以二次开发吗+

可以。它适合有 ESP32、Arduino、PlatformIO 或 Hermes Studio 使用基础的用户继续修改固件和交互逻辑。
